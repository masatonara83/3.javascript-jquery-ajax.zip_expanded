○問11 変数のスコープ(let)
・変数を宣言する際はvarを使用しますが、varを使用した変数はブロックの外
　でも使用できてしまいます
(例)
if (true) {
    var x = 5;
}
console.log(x); // 5と出力
・varをletに変えると変数のスコープをそれが使用されたブロック、
　文または式に限定することができます。
・動作を確認してください
